# experimental-gingivitis-metagenomics
Complete R code for metagenomic analysis used in published paper: Metagenomic characterization fo the supragingival and salivary microbiota during initiation and resolution of experimental gingivitis

## Repository purpose 

Taxonomic and funcational profiling:

* Importing MetaPhlAn and HUMAnN profiling datasets
* Alpha-diversity analysis
* Beta-diversity analysis
* Differential abundance analysis w. MaAsLin3
